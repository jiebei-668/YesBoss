# 自动触发记忆提取流程

## 流程说明
定时任务检查触发条件（interval/epoch_max），如果满足条件则自动触发记忆提取流程。

## 参与者
- 定时任务调度器
- TriggerService: 触发条件检查服务
- ChatMessageRepository: 对话消息仓储
- ResourceRepository: 资源仓储
- SnippetRepository: 记忆片段仓储
- PreferenceRepository: 偏好仓储
- CaptionGenerator: 资源描述生成器
- SummaryGenerator: 记忆摘要生成器
- EmbeddingService: 向量化服务
- MemoryExtractor: 记忆提取器

## 时序图

```mermaid
sequenceDiagram
    participant Scheduler as 定时任务调度器
    participant TriggerService as TriggerService
    participant ChatMsgRepo as ChatMessageRepository
    participant Extractor as MemoryExtractor
    participant CaptionGen as CaptionGenerator
    participant ResourceRepo as ResourceRepository
    participant SummaryGen as SummaryGenerator
    participant SnippetRepo as SnippetRepository
    participant PreferenceRepo as PreferenceRepository
    participant EmbeddingSvc as EmbeddingService

    Note over Scheduler: 定时触发（每分钟执行一次）
    Scheduler->>TriggerService: checkTriggerConditions()

    TriggerService->>TriggerService: checkIntervalCondition()
    Note over TriggerService: 检查是否超过interval时间间隔

    TriggerService->>TriggerService: checkEpochMaxCondition()
    Note over TriggerService: 检查未处理对话数是否超过epoch_max

    alt 满足触发条件
        TriggerService->>ChatMsgRepo: getUnprocessedMessages(interval, epochMax)
        ChatMsgRepo-->>TriggerService: List<ChatMessage>

        TriggerService->>Extractor: extractMemoryFromMessages(messages)

        Note over Extractor: 步骤1: 创建Resource
        Extractor->>CaptionGen: generateCaption(content, modality)
        CaptionGen-->>Extractor: caption

        Extractor->>EmbeddingSvc: generateEmbedding(caption)
        EmbeddingSvc-->>Extractor: embedding

        Extractor->>ResourceRepo: save(resource, caption, embedding)
        ResourceRepo-->>Extractor: resource

        Note over Extractor: 步骤2: 提取Snippet
        Extractor->>SummaryGen: generateSummary(resource.content, memoryType)
        SummaryGen-->>Extractor: summary

        Extractor->>EmbeddingSvc: generateEmbedding(summary)
        EmbeddingSvc-->>Extractor: embedding

        Extractor->>SnippetRepo: save(snippet, summary, embedding, resourceId)
        SnippetRepo-->>Extractor: snippet

        Note over Extractor: 步骤3: 更新Preference
        Extractor->>SummaryGen: updatePreferenceSummary(existingSummary, newSnippets)
        SummaryGen-->>Extractor: updatedSummary

        Extractor->>EmbeddingSvc: generateEmbedding(updatedSummary)
        EmbeddingSvc-->>Extractor: embedding

        Extractor->>PreferenceRepo: updateSummary(preferenceId, updatedSummary, embedding)
        PreferenceRepo-->>Extractor: updatedPreference

        Extractor-->>TriggerService: ExtractionResult

        TriggerService->>ChatMsgRepo: markAsProcessed(messageIds)
        TriggerService-->>Scheduler: 触发成功

    else 不满足触发条件
        TriggerService-->>Scheduler: 无需处理
    end
```

## 接口方法说明

### TriggerService
- `checkTriggerConditions()`: 检查触发条件
- `checkIntervalCondition()`: 检查时间间隔条件
- `checkEpochMaxCondition()`: 检查对话数量条件

### ChatMessageRepository
- `getUnprocessedMessages(interval, epochMax)`: 获取未处理的消息
- `markAsProcessed(messageIds)`: 标记消息为已处理

### ResourceRepository
- `save(resource, caption, embedding)`: 保存资源及其描述和向量

### SnippetRepository
- `save(snippet, summary, embedding, resourceId)`: 保存记忆片段

### PreferenceRepository
- `updateSummary(preferenceId, summary, embedding)`: 更新偏好摘要

### CaptionGenerator
- `generateCaption(content, modality)`: 生成资源描述

### SummaryGenerator
- `generateSummary(content, memoryType)`: 生成记忆摘要
- `updatePreferenceSummary(existingSummary, newSnippets)`: 更新偏好摘要

### EmbeddingService
- `generateEmbedding(text)`: 生成文本向量
