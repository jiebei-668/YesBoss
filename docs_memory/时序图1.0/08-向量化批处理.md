# 向量化批处理流程

## 流程说明
收集待向量化的数据（Resource、Snippet、Preference），批量调用LLM embedding API，批量更新embedding并存储到向量库。

## 参与者
- EmbeddingService: 向量化服务
- EmbeddingQueue: 向量化队列
- LlmClient: 大语言模型客户端
- VectorStore: 向量存储
- ResourceRepository: 资源仓储
- SnippetRepository: 记忆片段仓储
- PreferenceRepository: 偏好仓储
- MetricsCollector: 指标收集器

```mermaid
    Note over EmbeddingSvc: 记录处理指标

    end
```

## 接口方法说明

### EmbeddingQueue
- `getBatchItems(batchSize)`: 获取批量待处理项目
- `processEmbeddingQueue()`: 处理向量化队列

### EmbeddingService
- `processEmbeddingQueue()`: 处理向量化队列
- `categorizeItems(items)`: 分类整理项目
- `extractTexts(items)`: 提取待向量化文本
- `groupResourceByModality(resources)`: 按模态分组资源
- `groupSnippetByMemoryType(snippets)`: 按记忆类型分组片段

### LlmClient
- `batchEmbed(texts, model)`: 批量生成向量

### VectorStore
- `bulkInsert(indexName, ids, embeddings, metadata)`: 批量插入向量

### ResourceRepository
- `batchUpdateEmbedding(resourceIds, embeddings)`: 批量更新资源向量

### SnippetRepository
- `batchUpdateEmbedding(snippetIds, embeddings)`: 批量更新片段向量

### PreferenceRepository
- `batchUpdateEmbedding(preferenceIds, embeddings)`: 批量更新偏好向量

### MetricsCollector
- `recordEmbeddingMetrics(batchResult)`: 记录向量化指标

## 批处理策略

### 批量大小优化
```java
// 根据文本长度动态调整批量大小
public int calculateOptimalBatchSize(List<String> texts) {
    int totalLength = texts.stream()
        .mapToInt(String::length)
        .sum();

    // 假设API限制为8000 tokens
    int maxTokens = 8000;
    int avgTokensPerChar = 2;  // 粗略估计

    int optimalBatchSize = maxTokens / (totalLength / texts.size() * avgTokensPerChar);
    return Math.min(optimalBatchSize, maxBatchSize);
}
```

### 错误重试策略
```java
public class EmbeddingRetryStrategy {
    public List<float[]> executeWithRetry(List<String> texts) {
        int retries = 0;
        List<float[]> embeddings = null;

        while (retries < maxRetries) {
            try {
                embeddings = llmClient.batchEmbed(texts, model);
                break;
            } catch (EmbeddingException e) {
                retries++;
                if (retries >= maxRetries) {
                    throw e;
                }

                // 指数退避
                long delay = (long) (retryDelayMs * Math.pow(backoffMultiplier, retries - 1));
                Thread.sleep(delay);
            }
        }

        return embeddings;
    }
}
```

## 性能监控

### EmbeddingMetrics
```java
public class EmbeddingMetrics {
    private long timestamp;
    private String itemType;  // RESOURCE, SNIPPET, PREFERENCE
    private int batchSize;
    private long processingTimeMs;
    private int successCount;
    private int errorCount;
    private double cost;      // API调用成本
    private int tokensUsed;  // 使用的token数

    public double getProcessingRate() {
        return (double) batchSize / (processingTimeMs / 1000.0);
    }

    public double getCostPerItem() {
        return successCount > 0 ? cost / successCount : 0;
    }
}
```

## 队列管理

### 队列优先级
```java
public class EmbeddingItem implements Comparable<EmbeddingItem> {
    private String id;
    private ItemType type;  // RESOURCE, SNIPPET, PREFERENCE
    private Priority priority;
    private long timestamp;

    @Override
    public int compareTo(EmbeddingItem other) {
        // 优先级排序：Preference > Snippet > Resource
        int priorityCompare = other.priority.compareTo(this.priority);
        if (priorityCompare != 0) {
            return priorityCompare;
        }

        // 同优先级按时间排序
        return Long.compare(this.timestamp, other.timestamp);
    }
}

public enum Priority {
    HIGH(1),    // Preference: 影响面广
    MEDIUM(2),  // Snippet: 中等影响
    LOW(3);     // Resource: 影响面最小

    private final int value;
}
```
