# 时序图标准模板 v2.0

## 消息格式规范

```mermaid
 participant->>participant: InterfaceName::methodName(param1, param2)
```

## 核心接口方法映射

### TriggerService
- checkTriggerConditions()
- checkIntervalCondition()
- checkEpochMaxCondition()
- getUnprocessedMessages(intervalMs, epochMax)
- markMessagesAsProcessed(messageIds)
- countUnprocessed()

### ChatMessageRepository
- findUnprocessedMessages(intervalMs, epochMax)
- markAsProcessed(messageIds)
- countUnprocessed()
- findByIds(ids)

### MemoryService
- extractFromMessages(messages)
- queryMemory(query, topK)
- processBatch(messages)

### MemoryQueryService
- findResourcesByQuery(query, topK)
- findSnippetsByQuery(query, topK)
- findPreferencesByQuery(query, topK)
- findResourcesByPreference(preferenceName, query, topK)

### MemoryManager
- saveResource(content, conversationId, sessionId)
- createSnippet(request)
- createOrUpdatePreference(request)

### ContentProcessor
- generateCaption(content)
- generateSummary(content, memoryType)
- updatePreferenceSummary(existingSummary, newSnippets)
- batchGenerateCaptions(contents)
- batchGenerateSummaries(contents, memoryType)
- generatePreferenceName(snippets)
- generateInitialPreferenceSummary(snippets)

### EmbeddingService
- generateEmbedding(text)
- batchGenerateEmbeddings(texts)
- generateConversationEmbedding(conversation)

### VectorStore
- insert(indexName, id, vector, metadata)
- bulkInsert(indexName, vectorDataList)
- search(indexName, queryVector, topK)
- update(indexName, id, vector, metadata)

### ResourceRepository
- save(resource)
- saveAll(resources)
- findById(id)
- findByIds(ids)
- findBySessionId(sessionId)

### SnippetRepository
- create(request)
- createAll(requests)
- findById(id)
- findByIds(ids)
- findByPreference(preferenceId)
- findByResourceIds(resourceIds)

### PreferenceRepository
- upsert(request)
- upsertAll(requests)
- findById(id)
- findByName(name)
- updateStats(preferenceId, request)
