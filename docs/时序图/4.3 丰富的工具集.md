# 4.3 丰富的工具集

## 功能描述
总Agent和子Agent均被赋予强大的工具调用能力，包括但不限于文件操作、Blob处理、网络搜索，并全面支持各类型的MCP(Model Context Protocol)工具。

## 时序图

```mermaid
sequenceDiagram
    autonumber

    participant MA as MasterRunner/WorkerRunner
    participant TR as ToolRegistry
    participant NT as NativeTool
    participant MT as McpToolAdapter
    participant MCP as MCP服务
    participant SI as SandboxInterceptor
    participant TCT as ToolCallTracker
    participant LSM as LocalStreamManager

    Note over MA,TR: 工具注册与发现

    rect rgb(220, 240, 220)
        Note over NT,MCP: 系统启动时注册工具

        par 注册原生工具
            TR->>TR: 模块4::ToolRegistry::register(new FileReadTool())
            TR->>TR: 模块4::ToolRegistry::register(new FileWriteTool())
            TR->>TR: 模块4::ToolRegistry::register(new BlobProcessTool())
            TR->>TR: 模块4::ToolRegistry::register(new WebSearchTool())
        and 注册MCP工具
            MCP->>MT: MCP服务发现
            MT->>MT: 将MCP工具包装为McpToolAdapter
            MT->>TR: 模块4::ToolRegistry::register(mcpAdapter)
        end
    end

    Note over MA,LSM: Agent调用工具

    MA->>TR: 模块4::ToolRegistry::getAvailableTools(agentRole)
    TR-->>MA: 返回可用工具列表（包含工具描述和JSON Schema）

    MA->>MA: 调用LLM，LLM决定使用工具

    MA->>TR: 模块4::ToolRegistry::getTool(toolName)
    TR-->>MA: 返回工具实例

    Note over MA,SI: 沙箱安全检查

    MA->>SI: 模块4::SandboxInterceptor::preCheck(toolName, argumentsJson)
    activate SI

    SI->>SI: 检查工具名和参数是否在黑名单

    alt 安全检查通过
        SI-->>MA: 允许执行
        deactivate SI

        Note over MA,MCP: 执行工具

        alt 原生工具
            MA->>NT: 模块4::AgentTool::execute(argumentsJson)
            activate NT

            alt 文件操作
                NT->>NT: 读取/写入本地文件
            else Blob处理
                NT->>NT: 处理二进制数据
            else 网络搜索
                NT->>NT: 调用搜索引擎API
            end

            NT-->>MA: 返回执行结果
            deactivate NT

        else MCP工具
            MA->>MT: 模块4::AgentTool::execute(argumentsJson)
            activate MT

            MT->>MT: 将参数转换为MCP协议格式
            MT->>MCP: 调用MCP服务
            MCP-->>MT: 返回MCP协议响应
            MT->>MT: 将响应转换为统一格式
            MT-->>MA: 返回执行结果
            deactivate MT
        end

        Note over MA,TCT: 记录工具调用日志

        MA->>TCT: 模块4::ToolCallTracker::trackExecution(sessionId, toolCallId, toolName, argumentsJson, result, isIntercepted=false)
        activate TCT
        TCT->>TCT: 封装为InsertToolExecutionLogEvent
        TCT->>TCT: 投递到异步持久化队列
        deactivate TCT

        MA->>LSM: 模块3::LocalStreamManager::appendToolResult(toolCallId, result, isError=false)

    else 触发黑名单
        SI-->>MA: 抛出SuspendExecutionException
        deactivate SI

        MA->>TCT: 模块4::ToolCallTracker::trackExecution(sessionId, toolCallId, toolName, argumentsJson, "拦截", isIntercepted=true)
        activate TCT
        TCT->>TCT: 记录拦截日志
        deactivate TCT

        MA->>MA: 触发人机回环流程
    end

    Note over TR: 工具类型示例

    rect rgb(240, 230, 200)
        Note over TR: **文件操作类**
        Note over TR: - read_file: 读取文件内容
        Note over TR: - write_file: 写入文件
        Note over TR: - list_dir: 列出目录

        Note over TR: **系统操作类**
        Note over TR: - execute_command: 执行Shell命令
        Note over TR: - process_blob: 处理二进制数据

        Note over TR: **网络操作类**
        Note over TR: - web_search: 网络搜索
        Note over TR: - http_request: HTTP请求

        Note over TR: **MCP工具类**
        Note over TR: - mcp_database: 数据库操作
        Note over TR: - mcp_github: GitHub集成
        Note over TR: - mcp_slack: Slack操作
    end
```
